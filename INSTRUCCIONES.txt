========================================
        OF DOWNLOADER - INSTRUCCIONES
========================================

IMPORTANTE
----------
OF Downloader solo organiza contenido al que tu cuenta ya tenga acceso.
No evita suscripciones ni muros de pago.


TERMUX / ANDROID
----------------
1. Instala Termux desde F-Droid o GitHub, no desde Google Play.
2. Instala Termux:API desde la misma fuente.
3. En Termux ejecuta:

   pkg update -y && pkg install -y git gh
   gh auth login
   gh repo clone tacosandtypescript-debug/of-downloader
   cd of-downloader
   bash instalar-termux.sh

La primera instalación puede tardar varios minutos. Prepara Debian, Python,
FFmpeg y OF-Scraper.

Para abrir el menú:

   of

Para importar la cookie con el selector:

   of importar

Para probarla:

   of probar

Para probar si un perfil se detecta bien:

   of probar-perfil NOMBRE

Para elegir desde tus perfiles suscritos activos:

   of perfiles

La opcion 1 del menu hace lo mismo: lista perfiles activos, incluye gratis
cuando OnlyFans los devuelve, detecta posts/fotos/videos y pregunta antes de
descargar.

Logs visibles:

   /root/storage/downloads/OFBackup/ultima-descarga.log
   /root/storage/downloads/OFBackup/prueba-perfil.log
   /root/storage/downloads/OFBackup/perfiles-suscritos.log


LINUX
-----
1. En la terminal ejecuta:

   sudo apt update && sudo apt install -y git gh
   gh auth login
   gh repo clone tacosandtypescript-debug/of-downloader
   cd of-downloader
   bash instalar-linux.sh

2. Abre la app con:

   of

Las descargas quedan en:

   ~/Downloads/OFDownloader


WINDOWS
-------
1. Si no tienes Python 3.11 o 3.12, el instalador intentara poner Python 3.12
   automaticamente con winget.
2. Si no tienes FFmpeg, el instalador intentara ponerlo automaticamente con
   winget.
3. Si winget no esta disponible, instala Python 3.12 desde python.org y marca
   "Add Python to PATH".
4. Clona el repositorio privado:

   gh auth login
   gh repo clone tacosandtypescript-debug/of-downloader
   cd of-downloader

5. Ejecuta:

   powershell -NoProfile -ExecutionPolicy Bypass -File .\instalar-windows.ps1

5. Abre una terminal nueva y usa:

   of

6. Para abrir la ventana grafica:

   of-downloader

Para importar el archivo de la extension:

   of importar
   of probar

Las descargas quedan en:

   %USERPROFILE%\Downloads\OFDownloader


EXTENSIÓN DEL NAVEGADOR
-----------------------
Repositorio privado con Chrome y Firefox:

   https://github.com/tacosandtypescript-debug/of-downloader-browser-extensions

Descarga Chrome:

   https://github.com/tacosandtypescript-debug/of-downloader-browser-extensions/raw/main/artifacts/of_downloader_exporter-chrome-1.0.5.zip

Descarga Firefox:

   https://github.com/tacosandtypescript-debug/of-downloader-browser-extensions/raw/main/artifacts/of_downloader_exporter-firefox-1.0.6.zip

En Chrome:
1. Descomprime el ZIP.
2. Abre chrome://extensions.
3. Activa Modo de desarrollador.
4. Pulsa Cargar descomprimida.
5. Selecciona la carpeta que contiene manifest.json.

Si Chrome dice que falta el manifiesto, elegiste la carpeta equivocada.
Selecciona la carpeta exacta donde está manifest.json.


CONECTAR LA CUENTA
------------------
1. Abre OnlyFans en el navegador.
2. Inicia sesión.
3. Recarga la página.
4. Pulsa OF Downloader Exporter.
5. Exporta OFBackup-auth.json.
6. Cárgalo con of importar.
7. Comprueba con of probar.

Ayuda desde la app:

   of cookie ayuda

Para pasarlo entre móvil y PC, mueve OFBackup-auth.json como archivo:

   - cable USB
   - Google Drive
   - Nearby Share
   - correo propio como archivo adjunto
   - Telegram/WhatsApp como archivo, no como texto pegado

Si el selector no abre:

   of importar RUTA/OFBackup-auth.json

Envio directo sin mover archivo:

   of recibir-cookie --qr

La app muestra enlace rapido, URL local, codigo y QR seguro. En la extension usa
Enviar a OF Downloader y pega el enlace rapido. Funciona en la misma Wi-Fi o
hotspot y se apaga solo.

La app conserva solo:

   sess
   auth_id
   x-bc
   User-Agent

El resto del JSON se descarta.


ACTUALIZAR
----------
Desde el menú usa:

   8. Actualizar OF Downloader y reiniciar

O desde terminal:

   of actualizar-app

Manual en Termux:

   cd ~/of-downloader
   git pull
   bash instalar-termux.sh

Si Google Drive dice que falta rclone en Termux:

   proot-distro login --shared-home ofbackup-debian -- apt-get update
   proot-distro login --shared-home ofbackup-debian -- apt-get install -y rclone qrencode


SEGURIDAD
---------
- No pegues cookies en chats ni capturas.
- No publiques OFBackup-auth.json, config.json ni auth.json.
- Borra OFBackup-auth.json después de importarlo.
- Si una cookie se filtró, cierra esa sesión y genera otra.

Lee README.md para la guía completa.
