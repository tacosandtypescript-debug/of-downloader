========================================
        OF DOWNLOADER - INSTRUCCIONES
========================================

IMPORTANTE
----------
La aplicación solo descarga contenido al que tu cuenta ya tenga acceso.
No evita suscripciones ni muros de pago.


TERMUX / ANDROID
----------------
1. Instala Termux desde F-Droid o desde GitHub, no desde Google Play.
2. Abre Termux y entra en la carpeta de este proyecto.
3. Ejecuta:

   chmod +x instalar-termux.sh
   ./instalar-termux.sh

4. El instalador actualizará Termux, solicitará permiso para Descargas e
   instalará Debian, Python 3.13, FFmpeg y OF-Scraper.
5. Al terminar aparecerá el menú interactivo.

Para volver a abrirlo:

   of

Para descargar directamente con un enlace:

   of "https://onlyfans.com/ID/usuario"

La primera vez elige "Conectar mi cuenta". Puedes pegar una Cookie normal o
el JSON completo exportado por el navegador. El método recomendado es usar OF
Backup Exporter en Firefox y después ejecutar:

of importar
of probar

Si Android entrega un archivo vacío al selector, usa la ruta directa:
of importar ~/storage/downloads/OFBackup-auth.json

Elige OFBackup-auth.json en el selector Android. Este selector requiere el
paquete y la aplicación Termux:API; la aplicación debe proceder de la misma
fuente que Termux. OF Downloader conserva solo sess, auth_id, x-bc y User-Agent.

También puedes pegar el JSON entero, aunque ocupe varias líneas. Si solo pegas
cookies, después se solicitarán x-bc y User-Agent.


WINDOWS
-------
1. Instala Python 3.11, 3.12 o 3.13 y marca "Add Python to PATH".
2. Ejecuta iniciar.bat.
3. Introduce username, sess, auth_id y x-bc en la interfaz.


LINUX DE ESCRITORIO
-------------------
1. Instala Python 3.11, 3.12 o 3.13 y soporte para entornos virtuales.
2. Ejecuta:

   bash iniciar.sh


SEGURIDAD
---------
- No publiques config.json, auth.json ni OFBackup-auth.json.
- No envíes capturas que muestren sess, auth_id o x-bc.
- Renueva las credenciales si sospechas que alguien pudo copiarlas.
- Las descargas se guardan por defecto en Descargas/OFDownloader en Android.

Consulta README.md para ver todos los comandos y detalles.
